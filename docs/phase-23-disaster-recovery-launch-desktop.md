# Phase 23 desktop impact

Desktop launch evidence must cover Electron restart, hosted-app recovery, stale menu handling, subscription expiry during operation, printer unavailable, customer-display recovery and offline queue resynchronization. Support bundles remain redacted and should be attached to launch gate evidence instead of sent as unrestricted logs.

AI remains a manager assistance feature from the hosted web app. The desktop cashier shell must not auto-apply AI suggestions to prices, discounts, refunds, stock or staff actions. Any future cashier-facing suggestion must be clearly marked as optional and require an authenticated manager approval path.
